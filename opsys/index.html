Distro Recommendations

Here are my recommendations for Linux distros.
Please note that none of these pages/OSes are
related to or created by tredI9100.

Linux Mint

An overall great and user-friendly OS. I tried it once on a Thinkpad and I've used it on all my computers since.

Fatdog64

A small and lightweight distro, great for bootable USB drives and CDs.
You might have trouble running it on newer PCs though,
and updates don't happen very often.

Debian

A very robust and stable distro that a lot of other distros are based on, like Ubuntu.
New releases take a lot of time to be released, but this just means you don't have to upgrade to the new releases as much.



More things will be added to this list at some point.
